# Quizzilla

Quizzilla is my learning project where I'm building a Kahoot-style quiz app using TanStack Start, TanStack Query, HonoJS, Better Auth, Drizzle ORM, and more. My goal is to explore authentication, real-time interactions, and other key features while improving my coding skills along the way! 🚀
